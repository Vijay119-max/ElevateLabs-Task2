# Task 2 - Exploratory Data Analysis

## Objective
Perform Exploratory Data Analysis on the Titanic Dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data Inspection
- Missing Value Analysis
- Summary Statistics
- Histograms
- Boxplots
- Correlation Matrix
- Pairplot
- Survival Analysis

## Key Findings
- Females had higher survival rates.
- First-class passengers survived more frequently.
- Fare contains outliers.
- Age distribution is concentrated between 20-40 years.
